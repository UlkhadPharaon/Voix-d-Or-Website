# Studio Voix d'Or - Luxury Website

A pixel-perfect, cinematic website implementation for Studio Voix d'Or, featuring a "Gold & Void Black" aesthetic, glassmorphism effects, and fluid animations.

## 🎨 Visual Identity

- **Primary Colors**: Gold (`#D4AF37`) & Void Black (`#050505`).
- **Typography**: `Playfair Display` (Headers) & `Inter` (Body).
- **Effects**: Glassmorphism (`backdrop-blur`), Custom Gradients, Smooth Scroll, Micro-interactions.

## 🛠 Tech Stack

- **Framework**: React 18 + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM

## 🚀 Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## 📂 Project Structure

- `src/components/layout`: Navbar, Footer, Layout wrapper.
- `src/components/ui`: Reusable luxury components (Buttons, PageHeaders).
- `src/components/home`: Specific sections for the landing page.
- `src/pages`: Full page views (Home, Services, Portfolio, About, Contact).
- `src/lib`: Utilities (clsx/tailwind-merge).

## ✨ Key Features

- **Cinematic Hero**: Video background simulation with "Slow Zoom" effect.
- **Asymmetric Layout**: Services page with alternating text/visuals.
- **Masonry Portfolio**: Responsive grid for video showcases.
- **Scrollytelling**: About page with text reveal animations.
- **High-Ticket Contact**: Glassmorphic form with gold focus states.
